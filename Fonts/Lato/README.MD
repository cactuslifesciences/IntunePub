Official Lato fonts from Google Fonts, hosted for deployment via Microsoft Intune scripts.

## About

This repository contains the Lato font family for enterprise deployment purposes. The fonts are sourced from [Google Fonts](https://fonts.google.com/specimen/Lato) and are hosted here to facilitate automated installation via Microsoft Intune.

**We do not own the rights to these fonts.** All rights belong to their respective copyright holders.

## Font Information

- **Font Family:** Lato
- **Designer:** Łukasz Dziedzic
- **Source:** [Google Fonts - Lato](https://fonts.google.com/specimen/Lato)
- **License:** [SIL Open Font License (OFL)](https://scripts.sil.org/OFL)

## Included Font Files

This repository contains the following Lato font variants:

- `Lato-Black.ttf` - Black weight
- `Lato-BlackItalic.ttf` - Black Italic
- `Lato-Bold.ttf` - Bold weight
- `Lato-BoldItalic.ttf` - Bold Italic
- `Lato-Italic.ttf` - Regular Italic
- `Lato-Light.ttf` - Light weight
- `Lato-LightItalic.ttf` - Light Italic
- `Lato-Regular.ttf` - Regular weight
- `Lato-Thin.ttf` - Thin weight
- `Lato-ThinItalic.ttf` - Thin Italic

## Purpose

These fonts are hosted solely for internal deployment automation via Microsoft Intune. This repository serves as a centralized location for font distribution across managed devices.
