# Fonts

Official font collections hosted for enterprise deployment via Microsoft Intune scripts.

## Overview

This directory contains various font families sourced from publicly available font repositories. These fonts are hosted here to facilitate automated installation across managed devices through Microsoft Intune.

**We do not own the rights to any fonts in this repository.** All rights belong to their respective copyright holders and original designers.

## Purpose

These fonts are maintained solely for internal deployment automation purposes. This centralized repository enables:

- Consistent font availability across enterprise devices
- Automated deployment via Microsoft Intune
- Version control and update management
- Simplified font distribution workflow

## Adding New Fonts

When adding new font families to this repository:

1. Create a dedicated subfolder with the font family name
2. Include a README.md with proper attribution and license information
3. List all included font files and variants
4. Provide links to original sources
5. Verify license permits redistribution
