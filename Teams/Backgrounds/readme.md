# Teams Backgrounds

Custom Microsoft Teams background images for enterprise deployment via Intune.

## Overview

This collection contains branded and curated background images for Microsoft Teams. These backgrounds are deployed automatically to managed devices through Microsoft Intune scripts.

## Usage

Backgrounds are deployed to:
- **Windows:** `%LOCALAPPDATA%\Packages\MSTeams_8wekyb3d8bbwe\LocalCache\Microsoft\MSTeams\Backgrounds\Uploads`
- **macOS:** `~/Library/Application Support/Microsoft/Teams/Backgrounds/Uploads`

After deployment, backgrounds appear in the Teams background selection menu during video calls.
